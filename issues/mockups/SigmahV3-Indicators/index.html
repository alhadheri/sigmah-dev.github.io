v3.0 - Indicators


The new version of indicators management is planned for Sigmah v3.0 which should be released in 2017 as part of Sigmah Phase 2 project.


The mockups provided here are just preliminary draft versions to illustrate the planned features. The graphical design of those screens will be totally rethought by Sigmah usability expert.


Two scenarios have been mocked-up:

  1. Simple scenario (19 steps - ~3 minutes): this scenario uses the simplest configuration for indicators (no monitoring plan, simplest consolidation, etc.)

  2. Detailed scenario (49 steps - ~10 minutes): this scenario illustrates all indicators' available options but it is possible in the daily use of Sigmah to find an in between the detailed and the simple cases.
