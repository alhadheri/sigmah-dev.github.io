Summary0000928: Iteration "type" for more intuitive iteration group manipulation
DescriptionThe concept of "iterative group" is quite abstract. For end-users, creating, renaming and removing an "iterative group" is not very intuitive. Same for a "Name of the iteration" column in exports.

As a consequence, a new "Iteration type" field should be added when creating or editing an iterative group. This text field should be disabled by default in the group edition popup and become enabled only if the "Iterative group" checkbox is checked.

The "Iteration type" field content ITERATION_TYPE (ie: "training") will be used to rename the following elements:
- "New group iteration" popup title: 'New "ITERATION_TYPE"'
- "Iteration name (optiona)": 'Name of "ITERATION_TYPE" (optional)'
- "Rename iteration": 'Rename "ITERATION_TYPE"'
- "Remove iteration": 'Remove "ITERATION_TYPE"'
- "Group iteration rename" popup title: 'Renaming of "ITERATION_TYPE"'
- "Confirm deletion of group iteration" popup title: 'Confirm removal of "ITERATION_TYPE"'
- "Are you sure that you want to delete this iteration ?": 'Are you sure that you want to delete this "ITERATION_TYPE"?'
- "Iteration Name" column in exports: 'Name of "ITERATION_TYPE"'

If the "Iteration type" field is left empty, the labels will remain the current ones.
