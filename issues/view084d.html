Summary0000046: Default flexible elements for each project
DescriptionEach project, no matter its model, has some default flexible elements attached to it. These elements ensure that the basic properties of a project can be edited after its instanciation.

These default elements are the following:
- code (text field)
- title (text field)
- planned budget(number field)
- spend budget(number field)
- received budget(number field)
- start date (date field)
- end date (date field)
- manager (drop down list)
- organisational unit (drop down list)
