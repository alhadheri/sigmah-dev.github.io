Summary0000710: Sigmah not working without any message in Firefox if "Remember history" not allowed
DescriptionSince v2.0 (and its offline mode?), if you open Sigmah with a Firefox configured with "Never remember history" option (or in Firefox private mode), once you have logged in, the application won't load as it should do, and no message will come to explain the situation to the user.
=> he may have the feeling that Sigmah is bugged

An information message informing him that Sigmah cannot work with this History option, or in Private mode should be send to him.
Additional InformationFor more information about Firefox history options : https://support.mozilla.org/en-US/kb/settings-privacy-browsing-history-do-not-track [^]

This issue will be hard to fix because there is no sure way to detect that the browser is working in Private mode.
