Summary0000414: Project-category link no more working
DescriptionSince v1.1, it is no more possible to:
1. load existing projects which have links with a category
2. create a project model field linked to a category
Steps To ReproduceTo reproduce problem 2.:
1. create a new project model
2. create a new choice list field in this project model, and attach it to an existing category
=> when clicking on "Save", you will get an error message and your new field won't be created
Additional InformationAccording to the log file, this problem is linked with CategoryTypeDTO. The change which has occured in [http://code.google.com/p/sigma-h/source/detail?spec=svn1146&r=1114 [^] r1114] for 0000395 seems to have broken something.
