Summary0000109: Editable text document export of reports
DescriptionIn order to be printed or shared differently, the reports typed into Sigmah should be exportable into a text document editable format (either MS Word .doc, .docx or Word97 filtered HTML, OpenOffice Writer .odt, or .rtf).
Additional InformationA previous experience with the Dynamic COMPAS has shown that the export of HTML produced by the FCKeditor component could be reused without any big modification into the Word 97 filtered HTML format. For more information about this, please consult the PHP code relevant page of the Dynamic COMPAS at the following URL: http://compasdyn.svn.sourceforge.net/viewvc/compasdyn/trunk/cdyn-htdocs/project_document/create_custom_report.php?revision=319&view=markup [^]
