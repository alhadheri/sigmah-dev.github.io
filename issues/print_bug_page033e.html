Summary0000639: Add "Risks and Assumptions" for logframe Activities
DescriptionCurrently, it is not possible to describe Risks and Assumptions for Activities in the logical framework.
Some organisations need to be able to describe Risks and Assumptions for each of the Activity entered in the logframe.

A new option for Activity should be added: "Enable Risks & Assumptions".
This option must be set to "No" by default and has two possible values: "Yes"/"No".
