Summary0001057: Project tab closed before closure confirmed
DescriptionIf a modification is made in a project but not saved, and then the user decides to close the project tab, the tab will close before the user confirms or rejects the closure.
Thus, we reach a situation where the current tab is displayed as being the tab next to the current project but the content remains the current project (see attached screenshot).

2. Close the project tab
