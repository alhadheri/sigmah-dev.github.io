Summary0000044: Funding links between projects
DescriptionThe projects have now a budget. A project can be funded by other projects and can finance others itself.

These links are displayed in the dashboard page of each project.
A first grid, just below the phases panel, lists the projects which finances the current project (no matter of their types).
A second grig, just below the first one, lists the projects which are funded by the current project (no matter of their types).

The two grids provide actions to select or to create a new funding/funded project. When the new project is linked to the current one, the user must choose the amount of the funding (as a percentage of the budget).

The type of the project model doesn't interfer in these links. It's a distinct notion. The type is just reminded in the grids with the intended logos.
