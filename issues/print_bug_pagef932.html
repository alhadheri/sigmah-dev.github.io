Summary0000022: Undesired horizontal scrollbar hiding logframe title fields
DescriptionAn Undesired horizontal scrollbar hides logframe title fields.

This bug always happen in the following environment, whatever the screen size:
- local installation under Windows Vista
- Tomcat 6.0
- java 1.6.0_18
- PostGreSQL 8.4

But the bug never occurs while testing it on the Sigmah instance on the web provided by Ideia.

On Google Chrome 7.0.517.41, the bug has been reproduced both on the Ideia platform and my local platform, but only at the first load of the log frame page. On the second load, the logframe has been perfectly displayed both on the Ideia platform and my local platform.
