Summary0000573: Max number of files limit not respected from Reports & documents sub-tab
DescriptionIf a files list field has a max number files limit set up, it is still possible to add more files than the limit from the "Add file" button on the Reports & documents sub-tab.
Steps To Reproduce(i) create a field with maximum 2 file
(ii) on the management board when one file has been added “add file button” is unavailable (OK) but on the reports and document sub tab you can still add files to this tab trough add files button and then select the field.
(iii) the files will be added (which is uncorrect) and Not visible in the “management board” !
