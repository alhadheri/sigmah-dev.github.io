Summary0000992: No field for project out of main orgunit
DescriptionOn v2.2-rc1, if you reproduce scenario below, you won't be able to see fields of a project out of your main orgunit.
Steps To Reproduce1. set up your user account to be linked to two orgunit, ie: "Africa" as main orgunit, and HQ as secondary orgunit, always with a user profile with all privileges
2. create a minimal project model
3. create a project from this model into your main orgunit
4. open the project => the project is displayed normally with all required fields in Details sub-tab
5. change the orgunit of the project to an orgunit out of your main orgunit
6. close and reopen the project
