Summary0000240: Org unit cache list not correctly filled
DescriptionOrg unit cache list does not contain all the required org units when the connected user is not attached to the root org unit.
Steps To Reproduce
Additional InformationFor example: a user is connected to an orgunit "Asia", which contains 3 countries, and 2 bases for for the first country "Afghanistan".
Then, in the "Create project" popup and in the User modification popup, the user will only be able to see the first country "Afghanistan" and its 2 bases.
He should be able to create a project at the level of its attached orgunit ("Asia"), and to any other sublevel (like the countries "Kirghizstan" and "Pakistan").

As another consequence: it is not possible to create a user attached to the top Orgunit.
