Summary0000467: Data written in the indicator space can be moved on a near month
DescriptionWhen the user want to write a data in an indicator in the indicator entry view within a project, the data can be moved to the next month.
The issue come from the time zone of the user.
Steps To Reproduce- Change your timezone
- Open a project
- Go to the indicator entry tab
- Write a data in an indicator
- Save it
- Close and open again the indicator entry view for the same project
- Go to the month where you have written your data
- Your data is not here
- Go to the next (or previous) month and you will see that the data was moved to a bad month
