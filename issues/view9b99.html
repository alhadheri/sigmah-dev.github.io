Summary0000365: New warning message when project model has 2 groups with same vertical position
DescriptionIn the project/orgunit models management screen, some users feel a problem when they set by mistake the vertical position of a group to an existing vertical position of another group of the same container. (see the attached user report URL)

To prevent this problem, a user should have a warning message each time he makes an action which leads to have two groups of the same container with the same vertical position. When they will open a project/orgunit model, a check should be done to see if this problem occur, and if it does the message should then also be displayed.
This message could be for a project model: "The groups '#GROUP1_NAME#' and '#GROUP2_NAME#' of container '#CONTAINER_NAME#" are set to the same vertical position: one of them will not appear when you will create a project from this model.".
