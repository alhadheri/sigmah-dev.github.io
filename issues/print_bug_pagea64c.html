Summary0000936: End-user validation on fields to export to data framework
DescriptionTo give control on fields to export in a data framework, a new option should be added in project model to data framework mapping: "publication control".
If this option is set for a project model field mapped to data framework element, in all projects from this model, this field will not only display the data framework logo next to it, but this logo will open a drop-down menu in which the user can check or uncheck a "not for publication" checkbox.
Only users having a new "DATA FRAMEWORK PUBLICATION CONTROL" global privilege will be allowed to modify this checkbox.
