Summary0001072: Category selection list sometimes disabled since FF50
DescriptionSince Firefox 50 (or maybe even earlier), the category selection list in a choice list field edition popup is disabled each time you follow the "Steps to reproduce" scenario below.
The only solution then is to refresh the entire screen (F5 shortcut).

This bug has been reproduced both on Sigmah 2.2 and older Sigmah 2.0.2. So it seems linked only to some compatibility with latest Firefox version.
Steps To ReproduceIn a project model:
1. Create a first choice list and select a Category as "Related category"
2. Create a second choice list and select "No related category" as "Related category"
3. Try to create 3rd choice list => the "Related category" drop-down list is now disabled and you need to Refresh your screen and try again to have this related category enabled.
