Summary0000162: Indicators aggregated values management in Indicator Entry screen
DescriptionIndicators aggregated values should have different presentation in Indicator Entry screen.
An aggregated value should be displayed differently, in order that the user notices that it is computed and not a raw data. The tooltip on the value should be done accordingly.

When modifying an aggregated value, the user should get a warning message informing him that this modification will erase all the details which have been entered into the values participating in the aggregation. If he confirms, the value given for each of the values participating in the aggregation (VPA) will be computed on the basis of the new value given for the aggregated value spread in each VPA at the prorata of the old VPA in the formal aggregated value.
