Summary0001111: Unexpected error always when getting back online
DescriptionOn v2.3-beta, you always get an "Unexpected error" when coming back from the offline mode. There is no information given in the server log file.

And at the end of the synchronization process, you have always another error with this message : "Une erreur s'est produite en mettant à jour votre base hors-ligne de données. Vous pouvez continuer à travailler normalement, mais gardez à l'esprit que les données que vous consultez dans votre base hors-ligne de données ne sont pas toutes cohérentes avec le serveur. Mettez à jour votre base hors-ligne de données pour régler cela."
Steps To Reproduce1. open a project phase containing a file field
2. switch to offline (I used the Firefox offline mode)
3. attach a file to the file field
4. switch back to online your computer or firefox
=> you get the "Unexpected error" message
5. ask to "Connect & Synchronize".
6. at the end of the sync process, you get :
"Une erreur s'est produite en mettant à jour votre base hors-ligne de données. Vous pouvez continuer à travailler normalement, mais gardez à l'esprit que les données que vous consultez dans votre base hors-ligne de données ne sont pas toutes cohérentes avec le serveur. Mettez à jour votre base hors-ligne de données pour régler cela."
Additional InformationIssue also tested on v2.2-rc6 : the "Unexpected error is reproduced" but not the second error with the long message.
