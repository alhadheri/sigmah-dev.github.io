Changeset Added some corrections and optimizations for the management of the flexible elements (need an update of the data model):

- Rename the CSS styles name to be consistent with the existing styles.
- Added the functionality to mask the phases panel during its loading and saving.
- Added the removal functionality for files and files versions.
- Added the possibility to manage a files list as a required element.
- Added the 'multiple choice list' element (it's a question which allow to pick more than one answer).
- Changed the way lists of values of flexible elements are stored (files lists, triplets lists, questions lists). The old way was requiring a SELECT MAX() query to generate the new id of the list. The new way stores directly the ids of the elements of the list in the value field.
