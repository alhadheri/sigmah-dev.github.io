Summary0000507: Modification of locked content
DescriptionA new global privilege "Modification of locked contents" should be created to allow modification of:
- content of closed phases
- content of closed projects

Those modification can only change a value of a field, or add a file or set a value for an empty field. It cannot be used to set as empty the content of a field.
