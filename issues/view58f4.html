Summary0000168: Disable file & report creation menu item if parent phase is closed
DescriptionIn the "Reports & documents" sub-tab, when the user clicks on the "New report" button, a drop-down menu appears and the report elements belonging to closed phases should appear as disabled.
Currently, they appear as enabled, but when the user clicks on it, nothing happen.

Since v2.0, graphical behavior is the same for files in this "Reports & documents" sub-tab. So the new graphical behavior showing more clearly that closed phases files or reports items are disabled should be the same.
Advised behavior (with discussion with indianauthority97, thanks!) would be :
- mouse cursor style to normal arrow
- background color should be transparent (opacity to 0%)
