Summary0000981: Sigmah Central instances usage monitoring
DescriptionIn order to monitor the usage of instances running Sigmah on Sigmah Central, a solution must be found to track three key informations:
- last user session opening date
- number of active user accounts
- disk space use

At best, those informations will be available in a single file generated every day at 0:00 for all Sigmah instances hosted on Sigmah Central, and all those files will be available in a folder accessible by FTP.
Steps To Reproduce
Additional InformationQueries and command which could be used to collect the information on each instance:
- last user session opening date: "SELECT MAX(datecreated) FROM authentication WHERE true;"
- number of active user accounts: "SELECT COUNT(userid) FROM userlogin WHERE active=true;"
- disk space use: "du" shell command

The result could be a table like:

domain name | last session date | # active user accounts | disk space
urd | 2016-01-14 15:07:02.995 | 36 | 423MB


This solution could be possible through 2 shell scripts:
- a shell script available for each Sigmah instance which returns a row like above each time it is called
- a global shell script called everyday at 0:00 compiling the table by calling each of the former scripts, and which will generate the report file
