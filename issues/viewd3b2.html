Summary0001102: Impossible to load projects for new user if draft projects converted to normal projects
DescriptionOn v2.2-rc6, if you follow the scenario below, you won't be able to load all projects and will get an error message : "An error occurred while attempting to access the projects. Not all the projects were successfully retrieved."

The same scenario has been tested on v2.0.2 and there was no bug, so the issue seems to be new to v2.2.
Steps To Reproduce0. Create a draft project from a draft model
1. Set this draft model from draft status to Ready status
2. Create a new user with all privileges
3. Log in with this user
=> you get the error
