Summary0000064: Improve drop-down list default filtering behavior
DescriptionThe drop-down list used in the application has the following behaviors:
1. The user can start to type the first letters of the item he wants to select, and it automatically filter items which starts with the letters typed.
2. If the user select an item in the list, and then wants to change the value he has set, he needs to erase manually the text inside the field in order to be able to choose another item from the list.

Behavior 1 is nice, but if its consequence is behavior 2, then it would be better to have none of those behaviors.
At best, behavior 2 should be replaced by the ability to always be able to select an item from the full list. At worse the filtering feature of the drop-down list should be removed.
