Summary 0000078: Add a project and annual view in the calendar
Revision 2013-02-25 13:19 by osarrat
Description Some users have requested a new view in the calendar. A view in which they can see all activities planned for the project during the whole period of it.
To provide this feature, a new view should be added to the calendar: "project view". This view will show all calendars during a period starting at the beginning date of the project and ending at the ending date of the project.
If two elements overlap in this view (like 2 activities), each activity should have its own row.

If no end date is given for the project, the "project" view is not available, only the annual. And naturally, the "project" view is not available for orgunit.
