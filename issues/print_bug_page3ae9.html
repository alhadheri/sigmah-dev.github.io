Summary0000520: New date field option: saved in calendar
DescriptionIn the orgunit/project model management screen, add a option for the date fields: "saved in calendar", and choose to save either in the calendar or as a new reminder (pending or to do) or both.
When a value will be set for a field with this option, a new event will be added to the calendar (or saved as a reminder) with title of the field at the given date. If the value of the field is modified, the event (or reminder) in the calendar will be moved. And if the event (or reminder) is moved, the value of the field is modified (and of course, the modifications are historicised as any field).
Of course, if an event of reminder is linked to a date field which is locked (in a closed phase for example, or linked to a field that a user has no right to modify), this event or reminder cannot be modified by the user.
