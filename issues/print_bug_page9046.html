Summary0000292: Amendable project ending date freezes Details subtab
DescriptionIf a project model has in its Details subtab an amendable ending date (which is a common real life case), if the user locks the amendment, the Details subtab will never finish its loading and the screen will be impossible to use.
Steps To Reproduce1. Create a project model with an amendable project ending date
2. Give to this project the "Available" status
3. Create a real project from this project model
4. Set the ending date to any value
5. Locks the first amendment

=> There is now no way to modify the content of the Details subtab.
