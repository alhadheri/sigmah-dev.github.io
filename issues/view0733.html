Summary0000497: Set budget and start/ending date default fields as amendable by default
DescriptionIn most of the cases, if an NGO makes an amendment, the budget and start/ending dates fields will be part of the parameter of this amendment.
When creating a new project model, those fields should be amendable by default.

This feature should be set only for projects created from the version where it will be released. No need to set back old projects model with those default amendable status for budget and start/ending dates.
