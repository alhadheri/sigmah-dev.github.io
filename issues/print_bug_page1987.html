Summary0000266: Add a "New version" button in the Reports & documents sub-tab
DescriptionCurrently, there is no way to add a new version of an existing document in the Reports & documents sub-tab.

A new button "New version" should be added between "Attach a file" and "New report".
This button will drop-down a menu containing all the files which have already been attached to the project/orgunit, with indication of the phase and type as in the the "Attach a file" menu. If the user select one file in this menu, he will get a file selection dialog box to select the file he wants to attach, and finally a confirmation message like: 'Are you sure you want to attach the new file "XXX_v5.doc" as new version of file "XXX_v4.doc", defined as "TYPE_OF_FILE" in "SCREEN_OR_PHASE_NAME"?'
