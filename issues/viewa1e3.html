Summary0000527: Twin fields
DescriptionIn the project model management screen, add the possibility to set several locations in different containers to enter data for a field.

So far, when a user creates a data field, he can only select a single container where he wants his field to appear. But sometimes, it can be useful to let the user enter the information in the several locations, like a budget field which would be visible in the Details sub-tab and also modifiable in a phase.

This could be done through a new feature in model management: "Create a twin field". This feature will create like a duplicate of an existing field, and this "twin" can be set to be read-only or open for modification. This twin IS NOT a duplicate, it is just a kind of shortcut to the field.

If a location is in a phase which has been closed, as well as in another subsequent phase or in the Details sub-tab, then when this phase will be closed and the user will come back to it, he will see the version of the value of this field that was the current one at the time the closure of the phase.
