Summary0000993: Project orgunit choice list too limited if user not attached to HQ as main orgunit
DescriptionOn v2.2-rc1, if the user is attached to an orgunit different than HQ as main orgunit, the project orgunit choice list will be limited to organisation only member of his main orgunit. This situation appears even if the user is attached to the HQ as secondary orgunit. This behavior was not requested.

This orgunit choice list is limited in the same way everywhere: on contacts, on orgunit, etc. For simplicity, the orgunit choice list should contain all orgunits whatever is the attached orgunit of a user.
