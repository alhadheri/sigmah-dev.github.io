Summary 0000885: Field modification email notification to team profile
Revision 2016-07-18 12:51 by osarrat
Description Sigmah users request to be notified by email on some modifications on projects they are working on.

A part of the solution designed with some users is to add a new option in project models (not on orgunit/contact models) administration field edition popup : "Email notification on field modification to team members with profile(s)".
With this feature, it will be possible optionnally to assign one or more user profile(s) to a field. Each time the field will be modified, an email notification will be automatically send to all team members of the given project with one of the selected user profile.
If one of the assigned profile is selected as a profile as a whole as a team member, all users having this profile will be notified.

End-user information on fields with notifications [optional]
The attached bell icon could be attached to the field title having notifications. And if the user move his cursor above this bell icon, he could get a tooltip message informing him for each user profile for which notifications are planned: "Any modification made to this field will send a notification email to all project team members with profile 'NOTIFICATION_USER_PROFILE'.".
This end-user information is optional as part of the feature description.

Email notification user control [optional]
An optional added part of this feature is to give control to users on the notifications they received.
A new column "Email notification" will be added in the Team sub-tab members table. This column will be filled with a checkbox for each member.
For users having only the "View Team sub-tab" global privilege, they will be able to check/uncheck this checkbox for the row corresponding to them. If such a user is represented only by a "profile as whole" row in this team members table, he won't be able to change the notification configuration for this "profile as a whole" row.
For users having the "Edit Team sub-tab" global privilege, they will be able to edit the notification parameters for any member or whole profile part of the team.
