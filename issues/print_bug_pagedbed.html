Summary0000379: Multi-database
DescriptionIn order to strengthen absolutely the isolation of information among organizations using the same instance of Sigmah, adding a feature enabling the use of several databases is the ultimate solution.

To do so, a new boolean configuration property should be added: enable_multi_database .
If enabled, a new separate configuration file should list all databases available, and assign a common name to each one. This list could look like:
"Croix-Rouge Fran√ßaise" / #CRF_DATABASE_TECH_NAME#
"Secours Islamique France" / #SIF_DATABASE_TECH_NAME#
etc.

Enabling this property would add another field in the login screen : "Database". This field will be a drop-down list with all common names of databases of the new separate configuration file. This field should be after the password field.
To make the log-in process faster, a cookie with email and database name could be stored on the user computer if he wants (a new checkbox "Remember my login/database ?" would be added on the login screen for that purpose).
