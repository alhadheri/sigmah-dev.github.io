Summary0000534: Field-linked reminders incentives
DescriptionAs an incentive to use reminders to help users organize their work, the following feature should be added:
- In the project model management screen, each field location will have 3 new radiobuttons: (i)"Add incentive to attach 'to do' reminder", (ii)"Add incentive to attach 'pending' reminder" and (iii)"Add incentive to send email notification". (a textfield will be attached to (ii) to give a "pending reminder justification message" which will be displayed to the user)
- For each field with (i)'to do', a button "Plan reminder" will be added next to the field. When clicked, the user will start the creation of a new reminder which will be attached to the field, or select a reminder among all existing reminders.
- For each field with (ii)'pending', when the user will have filled the field, a popup with the "pending reminder justification message" will ask him if he wants to attach a 'pending' reminder linked to the data he just entered. (ie: to attach a pending reminder to set when a donor should reply to the proposal just attached).
 For each field with (iii)'email notification' option, the end-user will see a "Send notification" button close to the field. Clicking on this button will open a popup to create a reminder in which the user will have to give at least one recipient for an email notification (see 0000193).

Moreover, this feature will add to reminders an option to contain a link to the field which is attached to it. When clicking on the reminder, a popup will open, and in this popup where will be a message like: "This reminder is attached to the field 'NAME_OF_THE_FIELD'", and this name of the field will be an hyperlink which will open the project at the right location.

In order to keep track of the date when a reminder is closed (like when an installment is received from a donor), if there is a reminder incentive attached to a field in a selected group, it will be possible to attach the content of date or text fields to the the content of the reminder linked to their "neighbor" field. In this case, the date or text field will be read-only, and a message will inform the user than he must attach a reminder to their "neighbor" field to be able to fill this read-only field.
