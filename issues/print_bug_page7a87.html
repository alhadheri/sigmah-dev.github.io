Summary0001033: Missing new line in related project table after new project attached
DescriptionOn v2.2-rc2, if you add an existing project as a funding source (or funded project) to a project, the existing project won't appear immediately in the related project table as it should.
Steps To Reproducestep 1 -
From the dashboard of an NGO project, click on the « Attach » button in the « Funding sources » table.

=> A pop-up should open where you can select the project that you want to attach and specify the amount that the project is going to give to the NGO project.
        
step 2 -
Fill the pop-up and click on « Ok ».
    

=> A new line should appear in the table « Funding sources » with the project's code, title and the amount of money coming from the attached project.

Bug at this step.
