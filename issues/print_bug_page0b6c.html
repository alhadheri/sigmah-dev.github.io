Summary0000459: Cannot upload a new version of a file
DescriptionWhen an user wants to add a new version of a file to a file list in a project, an error may occured and stop the process of creating a new version of the file.
This error appears when an user wants to upload a new version of a file and when the last version of the file was deleted.
Steps To ReproduceCreate a project with a file list element.
Add to this element a file.
Add to this file some new versions.
Delete the last one.
Add a new file version again.
The error appears.
