Changeset - The administrator can create 'Draft Projects' for the preview (Click on 'Create draft project' menu below the 'Create Project' menu) with the possibility to delete them afterward. (Fixes issue 0000097)

- The notion of advancement is added to the logical framework activities. The user can update the percentage of progression of each activity by the 'Edit' menu (right click on the activity). The total activity progress impacts the project activity progress in the dashboard table. (Fixes issue 0000113)

- The sub-tab "Reports & documents" is added for the Organizational Units with the same behavior as for Projects. (Fixes issue 0000107)

- While importing a category if there is another category with the same UUID or GUID its elements will be updated. (Fixes issue 0000132)
