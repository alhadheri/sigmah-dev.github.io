Summary 0000399: Data frameworks list screen
Revision 2016-09-13 19:49 by osarrat
Description In order to manage data frameworks, an organisation should be first able to manage the list of data frameworks it is using. This screen will be available in the administration to all users having a new "List data frameworks" global privilege. Modification (edition, import, duplication) will only be allowed to those having a "Edit data framework" global privilege.
This screen should provide the following feature:
- create a new data framework
- delete a data framework
- duplicate an existing framework
- import and export a data framework (see child issue 0000133 for more details)
- open a data framework for modification (see 0000400)
- export matching content of each data framework (see 0000926 for more details)
