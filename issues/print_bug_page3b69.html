Summary0000491: Report field not working if attached to project Details sub-tab
DescriptionIn a project model, if a report field is attached to the project Details sub-tab:
- it won't be possible to create the report from the button "New" in the Reports & documents sub-tab
- once the report created from the Details sub-tab, it won't be listed in the Reports & documents sub-tab, and cannot be opened neither from the Details sub-tab
Steps To Reproduce1. attach a report field to the Details sub-tab of a draft project model
2. create a project test from this model
