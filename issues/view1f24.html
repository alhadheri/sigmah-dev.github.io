Summary0000384: Right to delete project
DescriptionMany users have reported the need to delete project.
Of course, such an operation should be available only to some users.

A new global privilege "Delete project" should be added.
For a user having a profile with this privilege, a dustbin button will be visible in the banner of all projects. If he clicks on it, he will get a confirmation message asking him to proceed, and if he does then the project will be no more available anywhere, and a feedback message will show up to confirm the deletion.
Additional InformationTo develop this feature extensively, be careful of the following visibility case: deleted projects which were attached to other projects should not be visible in the list of attached projects after being deleted.
