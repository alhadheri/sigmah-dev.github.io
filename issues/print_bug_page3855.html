Summary0000360: Impossible to set back a quantitative indicator value to empty
DescriptionIn the Indicator Entry sub-tab, once a user has saved a value for a quantitative indicator, he cannot set back this value to empty.
Steps To Reproduce1. Enter a value in a cell of a quantitative indicator in the Indicator Entry sub-tab.
2. Click on "Save".
3. Edit once again the same cell you've just entered a value in, and this time remove the value entered to make the cell empty.
4. Click on "Save".
5. Leave the Indicator Entry sub-tab.
6. Come back in the Indicator Entry sub-tab.
7. Browse to display the cell emptied.
=> The cell is no more empty, and is displayed with the value set initially.
