Summary 0000521: Aggregating linked projects calendars
Revision 2016-08-11 13:13 by osarrat
Additional Information The calendar list should use a treegrid and buttons to manage all the calendars to display. The screen which could be taken as an example is the "Manage organisational units" which uses a treegrid with checkbox, and has a toolbar with button above the checkbox.

There should be 4 "Calendar type filter": "Events", "Todo reminders", "Expected ", and "Activies". Those filters should be available as checkboxes in the toolbar (prefered solution), or as drop-down menu available from the toolbar.

The treegrid should have the following hierarchy:
- TYPE_ICON Project CURRENT_PROJECT_CODE - CURRENT_PROJECT_TITLE
  * CALENDAR_FULL_COLOR_ICON Events
  * CALENDAR_FULL_COLOR_ICON To do
  * CALENDAR_FULL_COLOR_ICON Expected
  * CALENDAR_FULL_COLOR_ICON Activities
- Related projects
  * Funding sources
    * TYPE_ICON Project PROJECT_CODE - PROJECT_TITLE
      * CALENDAR_LIGHTER_COLOR_ICON Events
      * CALENDAR_LIGHTER_COLOR_ICON To do
      * CALENDAR_LIGHTER_COLOR_ICON Expected
      * CALENDAR_LIGHTER_COLOR_ICON Activities
  * Funded projects
    * TYPE_ICON Project PROJECT_CODE - PROJECT_TITLE
      * CALENDAR_LIGHTER_COLOR_ICON Events
      * CALENDAR_LIGHTER_COLOR_ICON To do
      * CALENDAR_LIGHTER_COLOR_ICON Expected
      * CALENDAR_LIGHTER_COLOR_ICON Activities
