Summary0000586: Activate this phase should not disable the Save button
DescriptionBase on the scenario given below, the forced disabling of the "Save modifications" button when the "Activate this phase" action is performed should be removed.
"Save modifications" and "Activate this phase" should remain totally independant.
Steps To Reproduce1. When in one phase we enter some data
2. and then decide to click on “activate this phase” and accept the pop up.
3. Then the button “save modifications” can’t be clicked (stay grey)
4. and if we move to another phase... => all previously entered data are lost.
