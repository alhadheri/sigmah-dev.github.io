DescriptionBefore revision r988, when loading Sigmah in the root folder of the webapp will redirect the user to the ActivityInfo login.
On revision r988 (http://code.google.com/p/sigma-h/source/detail?r=988 [^]), a new behavior to redirect the user to the Sigmah login page has been implemented.
