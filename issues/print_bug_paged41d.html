Summary0000403: Full repartition of all values of qualitative indicators
DescriptionCurrently, only the major one full repartition of all values (and not only the major one).

For example, for a qualitative indicator with "Very bad", "Bad", "Good" and "Very good" as possible values, we could get "Good (40%)" as aggregated value, and a tooltip showing:
Very bad: 10%
Bad: 25%
Good: 40%
Very good: 25%

The aggregated value could be underline as an hyperlink, and the tooltip could be opened by clicking on it. It no hyperlink is given, the user won't guess that he can get more information about the value.
