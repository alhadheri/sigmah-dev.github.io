Summary0000635: Indicators values not in order in long period project exports
DescriptionFor project on a long period (more than 2 years for example), the values of an indicator are not exported correctly. In the exported spreadsheet, values do not appear in the column where they should do.
Steps To Reproduce- create a project
- define its start date to 01/01/2009, and its end date to 12/31/2015
- create a quantitative indicator
- create 3 sites
- give values to this indicator for the 3 sites for the following month: 01/2009, 04/2009, 07/2014
- export the indicators of this project

=> in the detailed tab of the exported spreadsheet for the indicator, the values of the indicator are not in the right column
