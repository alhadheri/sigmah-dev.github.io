Revision 2016-11-22 16:32 by osarrat
Description If you change some parameter like the name of an importation framework, the First row parameter will be set back to 0. First row parameter will also be set to 0, if you add a new variable to the importation framework.

First row parameter should not be changed when editing the general properties of an importation framework, or adding a variable to it.
