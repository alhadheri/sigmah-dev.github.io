Summary0000047: Open and display project reports
DescriptionPossibility for users to open and display reports attached to projects.

Reports are documents created from a "report model".

A report model is an arboresence of chapters. The numbering of the chapters is fixed and cannot be set up specifically for a model (see the mock-up to see the numbering chosen).
Each chapter can have zero, one or many flexible elements attached to it. Rich-text field and static labels are part of the flexible elements which can be commonly used in a report model.

Firstly, only rich-text fields will be available in report models.
