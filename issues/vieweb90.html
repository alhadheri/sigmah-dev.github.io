Summary0000558: Deleted versions of files should no more be available
DescriptionThere is a mean to download a deleted version of a file attached to files list field.
Deleted versions of files should no more be available.
Steps To Reproduce(i) Create two versions of a files on a "file list" field
(ii) open the history pop-up and delete the last version of the file (the 2nd one)
(iii) go back on your project. The displayed name of the file is the first version of the file (ok !) but
  - when you try to download it, it will download the 2nd version of the file that you deleted (instead of the 1st version).
  - open the history pop up and dowload directly the file from there. It will properly dowload the right version of file (1st one)
