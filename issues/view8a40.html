Summary0000576: Auto-export saved dates only in English format
DescriptionOn v1.2-rc3, the saved dates of the auto-export are only saved in English format. Even if the application is loaded in French, the auto-export backup will be presented with dates like "6.8.2013 12 AM".
Since the backups are listed in a chronological order, it is not a major issue. But it would be easier for the non-English users to have those dates presented in their native date format.
Additional InformationThis issue has been seen to target version=1.2 because it might be better to solve it before releasing the export feature.
If it won't be a burden to develop it in v1.3 or a later release (if for example there won't be an issue to migrate all the auto-export date format to the new locale-independant format), then the target version can be changed to 1.3 or 1.5.
