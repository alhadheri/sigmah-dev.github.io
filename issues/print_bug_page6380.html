Summary0000257: Impossible to let an indicator value empty after opening edition
DescriptionIn the Indicators Entry sub-tab, once a user has clicked on a cell, he cannot exit the edition mode and try to modify another cell.
The system doesn't let him not entering any value.

If he enters a value to escape, he will not be able to modify the value to let nothing afterwards.

It should be possible to let empty or to empty a cell of an indicator value.
