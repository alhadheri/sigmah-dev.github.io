Summary0000354: "br" & other strings added in the label of new fields in orgunit/project models
DescriptionWhen a user creates a new field in a project or organizational unit model, the text "br" is added in its label each time the label has more than one word. The label should be saved as entered by the user.

If the users tries to increase or decrease the font size, he will get "font size" before and "font" after the part of the string he wants to change the fond size.
If he tries to set the text to underline, the will get "span styletextdecoration underline" (same for italic and bold)
Questions marks "?" won't be kept in the string, as any other special character (<, -, etc.)
Additional InformationThis bug is linked with issue 0000221 which could have been only partly fixed.
