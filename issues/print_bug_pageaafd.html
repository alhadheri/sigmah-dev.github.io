Summary0000575: Auto-export schedule frequency improvement
DescriptionIn v1.2, the auto-export schedule can only be set to: Not Scheduled, Daily, Every 3 days, Every 9 days, Every 15 days.

An adoption focal point has reported that it would be more convenient to be able to set up the auto-export frequency to thinks like : every XX of each month, and/or every XX of each week.
