Summary0000686: While offline, Export all leads to a dead link page
DescriptionOn v2.0-rc1 while offline, if the user clicks on the "Export all" button in the Dashboard Projects table, it opens a new tab in his web browser which tries to open a URL without success.

The "Export all" button is not a feature which should work while offline, but the current behavior should be corrected because it leads to an error page.
Steps To ReproduceTwo solutions are possible to avoid this error:
1. hide the "Export all" button when offline (preferred)
2. display an information message popup "Feature unavailable while offline" when the user clicks on this button while offline.
