Summary0000860: Data framework feature foundations development
DescriptionData framework is new Sigmah concept. It is an evolution of the quality framework concept presented in feature 0000106 for example. Basically, there are models of projects models, or meta project models in other terms.
There are used to keep sure several project models share the same fields.

This issue is here to follow the development of the foundations of this feature:
- database schema to store those data frameworks, and link them to project models
- check the rule that each data framework element of a data framework implemented by a project model is actually mapped to a field
- check the rule that each field mapping a data framework element has the corresponding data type
