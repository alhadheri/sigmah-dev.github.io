Summary0000521: Aggregating linked projects calendars
DescriptionIn the calendar sub-tab of a project, all the events and reminders calendars of all the linked project should be available and disabled by default in the list of available calendars.
If a modification is set on the list of enabled calendars in this list, it should be automatically saved for the user.
The selection of types of calendars visible, or projects enabled should be eased to the user. Selection by types of calendar (reminder, events, activities...) for all projects should be possible. In the same manner, it should be possible to enable at once all linked projects, all source of funding, all funded projects.
In the calendar, the project code and orgunit code should be visible for all events/reminders coming from linked projects. If the user clicks on a code, it will open the project/orgunit accordingly. A strong work on the usability of this screen has to be planned.
Steps To Reproduce
Additional InformationThe calendar list should use a treegrid and buttons to manage all the calendars to display. The screen which could be taken as an example is the "Manage organisational units" which uses a treegrid with checkbox, and has a toolbar with button above the checkbox.

There should be 4 "Calendar type filter": "Events", "Todo reminders", "Expected ", and "Activies". Those filters should be available as checkboxes in the toolbar (prefered solution), or as drop-down menu available from the toolbar.

The treegrid should have the following hierarchy:
- TYPE_ICON Project CURRENT_PROJECT_CODE - CURRENT_PROJECT_TITLE
  * CALENDAR_FULL_COLOR_ICON Events
  * CALENDAR_FULL_COLOR_ICON To do
  * CALENDAR_FULL_COLOR_ICON Expected
  * CALENDAR_FULL_COLOR_ICON Activities
- Related projects
  * Funding sources
    * TYPE_ICON Project PROJECT_CODE - PROJECT_TITLE
      * CALENDAR_LIGHTER_COLOR_ICON Events
      * CALENDAR_LIGHTER_COLOR_ICON To do
      * CALENDAR_LIGHTER_COLOR_ICON Expected
      * CALENDAR_LIGHTER_COLOR_ICON Activities
  * Funded projects
    * TYPE_ICON Project PROJECT_CODE - PROJECT_TITLE
