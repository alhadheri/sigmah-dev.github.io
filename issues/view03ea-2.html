Summary0000523: Deleted files recovery page (garbage)
DescriptionA new page should be added to the Administration area: "Deletion recovery"
In this page, a part should be assigned to deleted files. It will be possible to administrator to delete permanently files sent to the garbage by users.. Or to recover them. The list of files will be given in a table with the following columns:
- File name
- File size
- File version
- Final version ?
- Deletion date
- Deletion author
- Field name
- Project/orgunit code
- Project/orgunit title
- Project/orgunit model name
2 new global privileges should be added: "See my own garbage" to give the ability to users with this right to recover the files they have deleted themselves, "Garbage admin" which allow to see everything and to delete permanently.
