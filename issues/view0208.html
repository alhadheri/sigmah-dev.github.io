Summary0000807: Impossible to import project models
DescriptionOn v2.0, it is no more possible to import project models between different Sigmah instances even if they are both in v2.0.

This bug occurs in particular when categories are linked to the exported project model.
Additional InformationIf you have two Sigmah instances in v1.2, you will able to import project models from one to the other. But if you upgrade both of them to v2.0, you won't be able anymore to exchange project models between them.
