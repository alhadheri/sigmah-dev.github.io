Summary0000859: New field type: linked projects field values list
DescriptionIn order to be able to view and edit data in linked projects directly from a project P, a new field type must be created. This is particularly important to give a comprehensive workflow vision within a single project for all users.

This new field type, linked projects field values list, will have the following options:
- project model or framework used for listing
- field of this project model/framework used for listing
- choice between "source of funding projects" or "funded projects" of the current project as source for listing
- view only or edition possible

For the end user, the field will look like a table of all values with the following columns:
- Value
- Project code (has hyperlink to the project itself)
- Project title
- history button

This field will work with those field types:
- file
- date, paragraph, text
- checkbox
- choice list
- reports
