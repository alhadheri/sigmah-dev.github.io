Summary0000110: Logframe Excel export
DescriptionIn order to share and print more easily the logical framework, it would be convenient to be able to export it as a MS Excel file (and optionnally under the OpenOffice Calc format as well)
Steps To Reproduce
Additional InformationIn the logframe view:
- column Risks and Hypothesis should be merged into a single one "Risks & Hypothesis" column as those 2 columns should be joined (see 0000189)
- indicators should be presented with their name and in brackets their value and the target value (ie: "Number of NFI kits distributed (1382/4500)")
- if the logframe is exported along with other project information in a full project export (see "Export at the project level" in Additionnal information of 0000388), then the name of the indicators could be transformed as hyperlink to the full definition of the indicator in the indicators tab

An empty logframe example is attached to give a basis for a general layout.
