Summary 0000020: Display and cinematic of the phase management panel
Revision 2010-10-25 12:30 by osarrat
Description The display and the cinematic of the active and the current phases could be improved on the screen "Project Dashboard".

Following improvements are suggested:
1. Display of the current phase
The current phase tab has a white background, but the background of the phase panel is light brown (even between the two components "mandatory fields" and "details of the phase"). The background of this panel should be white, to match the color of the tab.

2. Display of the active phase
The active phase is displayed in bold orange. It should be displayed in the normal style of the tab, but the bullet_star icon should be added at the left of the tab label to indicate that the phase is active.

3. Management of active phase
The button "Activate this phase" should be placed in the top-left corner of the "Details of the phase" box, and the bullet_star_new icon should be added to the button.
It the phase is active, the button "Activate this phase" should be replaced by "Close this phase". The icon to be used is bullet_star_black. When clicking on the button and if there are several phases which could be the next one, a menu should drop down from the button and the user should choose which phase to activate as next phase. In this menu, all items will have the bullet_star_new icon, and will have a label like "Activate phase 'Conception'", etc.

4. Display of the mandatory fields list
The icon used for empty field and filled fields should be replaced respectively by bullet_red and bullet_green

5. A warning message should be notified to the user to confirm that he wants to close a phase, informing him that the phase will become unmodifiable
