Summary0000425: Stored date is one day earlier than entered date on Sigmah Central
DescriptionOn Sigmah Central, when a user enters a project starting date, the date finally stored is one day earlier than the date entered (ex: "12/24/2011" instead of "12/25/2011").
Steps To Reproduce1. Open a project on Sigmah Central
2. Open the "Informations" sub-tab
3. Change the starting date
4. Click on "Save"
5. Close the project tab
6. Re-open the same project
7. Go back to the "Informations" sub-tab
=> the starting date is one day earlier than the day previously entered
