Summary0000114: Reminder form (edit & delete)
DescriptionIn order to edit and delete reminders, a reminder form should be added. This form should be opened by clicking on the name of a reminder which should be underlined as an hypertext link.
In this form, the user would be able to:
- update the description of the reminder
- update the date of the reminder
- change the status of the reminder
- delete the reminder (deletion will just hide the reminder)

 This feature is almost perfectly developed, congratulations!
I've just seen three little remaining imperfections, only on feedback messages:
- The feedback message to inform the user after a modification of a single todo reminder is using plural instead of singular. The message should be transformed from "The todos have been correctly modified." to "The todo reminder has been correctly modified.".
- There is no feedback message to inform the user after a modification of a "Pending" reminder.
- The feedback messages used for modification and deletion of a reminder should not be the same. For example for a todo reminder, instead of using the message "The todos have been correctly modified." for deletion, the following message should be used "The todo reminder has been correctly deleted.".
