Summary0001021: Missing field code in v2.1=>v2.2 migration new fields creation
DescriptionOn v2.2-rc1, there is no field code added for new field created in the v2.1=>v2.2 data migration. And since it is not possible to edit field codes in maintenance mode (see 0001020), users can't include new fields created in that migration into computation fields formulas.
As a consequence, default field codes should be added in the SQL migration script.
