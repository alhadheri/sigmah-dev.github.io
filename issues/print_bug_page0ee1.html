Summary0000800: Report Word export exported with a spreadsheet file extension
DescriptionOn v2.0-rc5, if you export a report with the "Export Word" button, the file you will download will have the "Project synthesis spreadsheet file format" extension (.ods/.xls).

If you change manually the extension of the downloaded file to .doc, it will open normally in MS Word and will have the right content. This bug is just about the file extension.
