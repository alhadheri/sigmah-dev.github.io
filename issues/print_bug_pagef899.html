Summary0000359: Indicators values coming from Datasources not included in aggregated value
DescriptionEven if an indicator has several datasources, the values coming from the other datasources are not included when computing the indicator aggregated value.
Steps To Reproduce1. Create a logical framework with a quantitative indicator I1 in project A (2 sites created in project A).
2. In the Indicators Data Entry tab of project A, set a value for indicator I1, and save.
3. Copy-paste this logframe from project A to project B, and accept to link the I1 indicator duplicated version to the source indicator I1
4. Open the Indicators Data Entry tab of project B, and modify the value of indicator I1 for a site specific to project B, and save. (the sites and value set for I1 in project A will be visible in this tab)
5. Open the Indicators Management screen of project B: the displayed aggregated value of indicator I1 will only sum up the values set in project B, and not all values coming from all datasources of the indicators.
