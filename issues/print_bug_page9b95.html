Summary0000642: New field type: image file
DescriptionAs reported by a user (See User Report URL), a new field type would be useful: image file.
With this type of field, and end user would be able to load images into Sigmah and see directly the image uploaded when loading the page of the field (no need to download the file).
Steps To Reproduce
Additional InformationThis field type would accept only one file per field, but can accept several versions. The field may look like a single file field with an image preview below.
If the image is very large, the image below will be clickable so that a user can see the full image by clicking on the preview.

To make the field easy to use, we could limit the allowed image file types to .png and .jpg.
