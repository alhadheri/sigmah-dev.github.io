Summary0000836: Linked projects aggregation field
DescriptionSome users have requested to have a new field in their project models: a field for aggregation of fields from linked projects.

When designing a project model, the administrator can create fields from a new type: Linked projecst aggregation.
If he selects this field type, he will have to define:
- the type of aggregation needed: Mean or Sum (other aggregation methods may be added later on)
- the project models which will be taken into account in this aggregation
- for each project model, the single Number field to be used as data for the aggregation
- the type of linked projects to take into account: only source of funding projects, only funded projects, or all linked projects (both source of funding and funded projects)

Then, when a project will be created out of this project model, as a value for this field, the user will see the aggregation of:
- for all projects which are directly linked to this orgunit with the project model and the type of linkage as parametrized in the aggreation field
- the mean or sum of all fields described in the detail of the aggregation field of those projects
