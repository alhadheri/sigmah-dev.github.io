Summary0000717: Confirmation message on session closure
DescriptionEven if the introductory offline popup warns the user that he should not close the session if he wants to be able to continue to use Sigmah offline, the best solution for this warning would be a confirmation message to inform him about this risk when he tries to close session.

When the user clicks on the "Close session" button, he should get a confirmation popup like "Once you have closed your session, you won't be able to use Sigmah offline until you reopen a session online. Are you sure you want to close your session?".
