Summary0000208: Changing of Project's OrgUnit
DescriptionIn order to better manage project's organizational unit in relation with indicators' sites, the following change are required:
1. the project model default flexible element "Country" should become read-only (to be used for display in the banner or anywhere else): it should not be possible to change the country of a project, only the organizational unit should be changeable
2. when modifying a project's organizational unit:
(i)if no indicator has yet been created for the project, the project's organizational unit can be changed without any notice
(ii)if indicators have yet been created for the project, the project's organizational unit can be changed but the user will have to confirm after reading that the project will remain attached to the current country for indicators' sites management and not the country linked to the new OrgUnit (this message appears of course only if the 2 countries are different)
