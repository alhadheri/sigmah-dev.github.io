Summary0000926: Data framework matching content general export
DescriptionData framework can be used to export to a standard format information stored and structured differently in Sigmah. For that purpose, a flexible export feature should be developed.
In the data framework edition screen, it should be possible to attach a data framework content export plugin file. This file could be for example a Jar file. When attaching this file, the system should validates that the file has the correct format, and that it does match with the data framework structure. Once the plugin file is attached, the "Export matching content" button in the Data frameworks list screen will be enabled. When this button will be clicked, Sigmah will export all content of the fields mapped to the data framework through the plugin.

An option for the format of this plugin file that should be explored in depth is the open source Job Script file from Talend Open Studio for Data Integration. This open-source ETL solution has already an extensive range of connectors, and it could provide a very powerful solution to build upon.
