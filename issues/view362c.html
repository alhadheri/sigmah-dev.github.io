Summary0000276: Add a "Groups" tab in project/orgunit model management
DescriptionCurrently, when a user creates a group, he cannot access it anywhere until he attaches it to a field.
This should be corrected by creating a new tab in the project/orgunit model management screens: "Groups". In this tab, the user will have a table with all the details of the groups created for this model.
