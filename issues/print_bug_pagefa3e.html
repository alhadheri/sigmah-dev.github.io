Summary0000932: "Select contact" button in "Possible duplicates found" popup
DescriptionIn order to clarify the behavior of contact duplicates management, the "Update contact" button in the "Possible duplicates found" popup should be replaced by a "Select contact".

Clicking on this "Select contact" will open the same second screen of the "Possible duplicates found" popup in which all differences between the contact to create and the selected possible duplicate are listed.
This second screen should as well be modified as following:
- a longer header message should be displayed above the list of fields like in the first screen of the popup: "Please see below all differences between the existing selected duplicate and the contact you wished to create. If you agree with all following modifications, click on the 'Update existing contact' button. If you prefer to keep the existing contact as it is, click on 'Select contact without modification'. Finally if you want to select another duplicate, click on 'Select another duplicate'.
- two new buttons: 'Select contact without modification', 'Select another duplicate'
- rename button 'Update contact' into 'Update existing contact'
