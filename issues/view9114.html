Summary0000684: Error message when trying to open offline a project not yet in offline data base
DescriptionOn v2.0-rc1 while offline, when a user has not yet opened online any project in his session and tries to open a project (favorite or not), he gets a confusing error message: "You are not authorized to access this resource.".

He should not get an error message but an information message instead, because there is actually no error.
And the message should not be about authorization, but more "You cannot open this element offline, because this element has not been downloaded in your offline data base".
Additional InformationSame error message which should be replaced by the same Information message when trying to open a sub-tab not available offline because not downloaded in the offline data base.
