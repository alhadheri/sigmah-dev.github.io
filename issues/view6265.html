Summary0000977: Data framework matching content single project export
DescriptionIn order to export a specific project in the format of a data framework, a new option should be added at the data framework configuration level: "Exportable at the project level". An additional optional parameter can then be set: "Viewing privacy group for exportation at field level".

If this option is set, for all projects of all project models implementing the data framework, there will be an additional button next to the "Delete" and "Excel Export" button: "Export to...". This "Export to..." button will be a menu button opening a menu with as items the names of all data frameworks implemented by the project model.
If the end-user has the required viewing privilege on the privacy group set at the data framework level, he will be able to select the corresponding item in the menu and exports this project in the data framework format.
