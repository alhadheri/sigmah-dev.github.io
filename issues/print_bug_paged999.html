Summary0000439: Project/orgunit model group edition popup doesn't load group position
DescriptionWhen the user loads the group edition popup in a project/orgunit model edition screen, the group position is not loaded in its drop-down list.
Steps To Reproduce1. Open a project model (scenario could be reproduced the same way with an orgunit model)
=> The Project model edition screen is opened on the "Fields" tab.
2. Click on a group hyperlink in the rightmost column of the list of fields.
=> The Group edition popup is opened but there is no value loaded for its attribute "vertical position".
