Summary0000563: Message error when loading project with empty category value
DescriptionIn a question element linked to a category, if a value is given to the question element in a project and it is removed, an error message occurs when the projects are loaded in the dashboard.
Steps To Reproduce1. Create a category with different values
2. Add a question element to a project model linked to the category
3. Create a project linked to the project model
4. Check a of the question's choices
5. Save the project
6. Uncheck all of the choices of the question
7. Save the project
8. Go to the dashboard and refresh the projects' list

Waiting for the fix, such values can be deleted from the 'value' table with no side effect.

Here is the MySQL query to select the empty values of question_element linked to a category:

SELECT `id_value`
FROM `value` AS v LEFT JOIN `question_element` AS q ON v.`id_flexible_element` = q.`id_flexible_element`
WHERE v.`value` = "" AND q.`id_category_type` IS NOT NULL

You can safely delete the selected values.
