Description On v2.2-rc6, if you attach first a user to a sub-orgunit, and then to the root orgunit, you won't be able to load all projects and will get an error message : "An error occurred while attempting to access the projects. Not all the projects were successfully retrieved."

The same scenario has been tested on v2.0.2 and there was no bug. Since v2.1 hasn't changed anything on orgunit, the issue seems to be new to v2.2.
