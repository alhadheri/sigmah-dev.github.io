Summary0000991: No deleted project models in Global export configuration
DescriptionCurrently, the deleted project models appear in the global export configuration.
Global export configuration should have all used and unavailable project models which are not deleted.
Steps To Reproduce1. in the Administration, create a project model
2. set its status to "Available"
3. from the Dashboard, click on the "Export all" button to open the Global export of projects popup
4. click on the "Change configuration" button in this popup
=> your new project model is here
5. go back in the "Project models" section of the Administration
6. delete your new project model
7. go back on the "Global export of projects configuration" popup
=> your new project model is still here (and it shouldn't be)
