Summary0000103: Edit title banner of the OrgUnit "Organigram" block
DescriptionCurrently, in a organization where HQ would be named "HQ(Siège du CAM)", the main Dashboard organigram component would for title "HQ(Siège du CAM) : Organigram".

If the organigram component displays the whole organigram of the organization, the title of the block should be: "Organigram - #NAME-OF-THE-ORGANIZATION#" (ie: "Organigram - Comité d'Aide Médicale").

If the organigram component displays the organigram of a specific organizational unit of the organization (even the top one which is usually the headquarters), the title banner should become: "Organigram - #CODE-OF-THE-ORGUNIT#(#NAME-OF-THE-ORGUNIT#)" (ie: "Organigram - HQ(Siège du CAM)").

This change should be applied on the Main Dashboard screen and all the screens where this block will appear.

osarrat   
2011-05-09 16:46   
One information which is missing for you is where this component can be displayed. This orgunit organigram block can be displayed in the main dashboard, and in the panorama subtab of each orgunit page. To open an orgunit page, you actually need to click on a link of an orgunit in any of these organigram components. By opening different orgunit, you will see different appearance of this component, always with a different subset of orgunit to display .


The top orgunit to display in an organigram component doesn't depend directly on the user connected, but on the contextual orgunit of the current screen. So, if the opened screen is the main dashboard , then you have to display the organigram for the whole organization. Is it clearer?
