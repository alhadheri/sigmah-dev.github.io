Summary0000581: Limit Project synthesis export to visible fields to user
DescriptionIn v1.2, the Project synthesis exports contain all fields set as "Exportable" in their project model, whoever is the user clicking on the "Export" button and whatever are his rights to view information on fields. To protect a field from being exported, the only solution is not to set it as "Exportable" for all users.

It would be better if there would be an extra automatic filter on Project synthesis export which would automatically exclude all fields which are not visible to the current user.
If this feature is developed, an extra information should be added in the Project synthesis export like "This Project synthesis is targeted to user of profile X, and should maybe not be shared with everybody".
Steps To Reproduce
Additional InformationWhen this feature will be developed, the information about the "Exportable" parameter of fields should be updated accordingly in the User guide : http://www.sigmah.org/node/422 [^]
