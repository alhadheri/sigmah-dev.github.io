Summary0000631: Add a message to create a first Site if none on Indicator Enty opening
DescriptionMany users are confused not to be able to enter values for the indicators they have created. They don't know that they need to create a site before being able to enter indicator values.

If at least one indicator is created but no site for a project, a message informing the user that "You must create at least one site in the Map sub-tab before being able to enter values for your indicators" should be added.
