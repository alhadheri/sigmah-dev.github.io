Summary0000544: Calendar empty if user timezone different from server timezone
DescriptionThe calendar is totally empty if the user timezone is different from the server timezone.
Steps To Reproduce1. create some data in the calendar of a project
2. open the calendar of the project
=> you can see what you've created
3. close the project
4. change the timezone of your computer
5. open the project back
6. open the calendar
=> there is no more any data visible in the calendar
