Summary0000513: Budget Excel export improvement
DescriptionIn excel exports, project budget is now presented only with a cell containing a text for the ratio like "20.0 / 100.0". To help end users to process this content, and to be consistant with the "Flexible budget field" feature, the budget should be presented with one separate cell for each component of the budget as defined in the flexible budget field. There could be a cell for "accepted budget", "planned budget", etc.
