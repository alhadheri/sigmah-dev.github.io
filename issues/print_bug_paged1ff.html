Summary0000330: Anyone can reset any one else's password
DescriptionIf you click "forgot password" on the login screen, and type in anyone's email address, their password will be immediately changed and sent without authenticating the request. This has created some confusion.

Rather a link with a secure key should be sent to the user's registered email address, which the user can use to reset their password if the request was authentic.
Steps To Reproduce
Additional InformationThis behavior is truly dangerous.
2 behaviors seem possible for password lost:
- the behavior described by Alex with an email which can lead to a change (through a link with a secure key)
- or a simple email with a reminder of the current password of the user with no change (in the email, the user can be informed that he need to ask his Sigmah administrator to update his password if he prefers to change it)
