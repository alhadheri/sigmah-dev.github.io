Summary0000640: Technical export to IATI standard
DescriptionIATI stands for "International Aid Transparency Initiative". Basically, it is a XML standard used to bring more transparency on aid projects.

More and more Sigmah users request to be able to export information to IATI standard. Such kind of export is planned as part of the development of the "framework" concept used in the General revision of indicators management (0000532).
But the need to export to IATI is quite urgent for some organisations, and it would be nice to have a basic, even using a command line interface, solution to export an organisational Sigmah database or a chosen subset of it into the IATI standard.
Two export kinds might be expected in this basic tool: export of all projects of an organisation, or export of a single project.
Steps To Reproduce
Additional InformationTwo options are imagined so far to develop this feature:
- either a tool which will convert the content of the database into a IATI XML file
- or a tool which convert an .xls/.ods global export file into a IATI XML file

For more information, take a look at IATI standard developer documentation: http://iatistandard.org/201/developer/ [^]
