Summary0000910: Improve contact list field usabilty
DescriptionOn v2.2-beta1, the contact list field is not respecting the mockup http://www.sigmah.org/issues/mockups/CRM/#projet_page [^] . This mockup has been provided after the requirements document, and is now the basis to use to give the best usability to users.

Some missing parts have a strong effect on user experience, some other have a weaker importance.
High importance :
- no hyperlink on contacts selected as part of the contact list: just as related projects or orgunit, each contact selected as attached to a project should appear as an hyperlink to the related contact page.
- no "Import" & "Export" button (expected only from v2.2-rc1)
- add new contacts directly through a button, and not from a drop-down list: this lonely button on the mockup was meant to open a popup to search&select a contact, or create it ; the drop-down list option with all contacts will soon be hard to use with several hundred contacts (the search option of this "missing part" is not actually presented in the CRM requirements document, and can be considered as a feature request)

Low importance:
- same layout as files list field: brown title header, toolbar, grid...
- details on each contact: same columns to present contacts as on the Dashboard Contacts list

Here are my suggestions of improvement of the "Select a contact" popup:
- let's move the "Create" button on the toolbar of the contact list (and remove the wheel icon for the "Create" button), and dedicate this popup to the selection of an existing contact
- add the event of pressing the "Enter" key when typing a text to search for a contact as a way to start the search
- add a notification message when no contact is found for a search like: "No contact with this search term."
- add a "Select" button at the end of each result row, which will select the contact and add it to the contact list
- add an "Contact identifier" column at the extreme left of each row
- replace the "Add" button on the bottom of the popup by a "Cancel" button
