abertram (developer)
2011-06-07 08:49

 - Centering on the right country requires correct bounding boxes (X1,Y1,X2,Y2) in the County table.

- The demo.sigmah.org deployment does not have a valid google maps api key, so this will cause some problems

- for sites, I have changed the default ActivityInfo behavior to correspond with your design: the map is reflects the selected indicator, so if a site does not have values for a given indicator is not shown. You can select different indicators to map by clicking on the icon.

I think I prefer the AI design, which shows all sites, and allows you to have the grid and map side-by-side so that you can drag and drop between the two. Ideally we should have another project tab "Cartographie" for the management of sites, but in the short term we could tear out the map icon selection and just show all sites on the map.
