Summary0000486: Field cross-checkable Excel global export
DescriptionWhen exporting several projects, the same fields for different projects in different project models are not easily cross-checkable. It is a limitation for comparing data at the organizational level.
A solution to fix should be found.
Steps To Reproduce
Additional InformationField cross-checking suggested solution:
-At the project model level, when designing a field, you can check a box telling “this field should be cross-checked with...”.
-If checkbox is checked, the system automatically will look for fields in other project models which have the same name.
-The admin can then select from the suggestion or any other project model field with which field this new field should be cross-checked.
