Summary0000546: User edition popup doesn't load user invitation language and password
DescriptionWhen you want to edit an user (e.g. give him new profiles) you always have to change the invitation language (which isn't properly loaded).
Much more serious, since the password is neither loaded, if the user clicks on save without entering the password, it will empty the password... and allow to enter the application just with an email without entering any password!
Steps To ReproduceScenario for the password part of the issue:
(i) add a new user. The user can be connected with its own password
(ii) Edit the user in "users" screen. Change only the profile without changing the password or even change nothing just save.
(iii) Logout and try to connect to the user : you can connected with no password or any wrong password.
