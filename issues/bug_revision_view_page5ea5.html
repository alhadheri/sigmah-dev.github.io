Summary 0000129: Sigmah instance registering system
Revision 2015-02-05 16:01 by osarrat
Additional Information The information message explaining to the user the reason of this registration could be:

French message
Sigmah est développé avec le soutien financier de bailleurs de fonds (principalement), les contributions annuelles des membres de sa Coopérative de Pilotage, et l'aide cruciale de volontaires. Afin de continuer à améliorer ce logiciel, nous devons être en mesure de fournir aux bailleurs quelques données simples sur la diffusion de Sigmah.

IMPORTANT: Projet de logiciel libre dirigé par un groupe ouvert d'ONG qui l'utilisent elles-mêmes, Sigmah n'apprécie pas les logiciels qui partagent des informations sur l'utilisateur. Celles à renseigner ci-dessous sont les seules qui vous seront demandées. Elles seront les seules transmises dans une base protégée sur le serveur du projet www.sigmah.org (code source public de Sigmah faisant foi).

Par conséquent, à chaque démarrage des sessions d'administrateurs jusqu'à ce que vous ayez pu prendre le temps de le remplir, le formulaire suivant vous sera proposé.

English message
Sigmah software is developed with the financial support of donors (mainly), yearly contributions of the Steering Cooperative members, and the crucial help of volunteers. In order to continue to improve the software, we need to be able to provide to donors few basic data about Sigmah dissemination.

IMPORTANT: Free Libre Open Source Software project managed by an open group of NGOs using it themselves, Sigmah doesn't like software which share information about the user. The informations to fill below are the sole which will be requested to you. They will be the unique information that will be send to a protected database on www.sigmah.org (public source code available).

Consequently, at each startup of an administrator session until you have time to fill it up, the following form will be shown to you.





Extra informations to collect
Later on, through this system, we might also track:
- the number of organization on the instance
- the number of users per organization
- the number of project models per organization
- the number of organizational unit models per organization
- the number of organizational units per organization

To be confirmed... and then, this information might be given to the user in the resgitering popup.

After discussion with some members of the Steering Cooperative, it has been decided not to share those usage information.

The decision to develop this feature has been validated in the Steering Cooperative meeting of the 30th May 2012.
